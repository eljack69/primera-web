# Demo Landing — FlujosIA

Landing estática de demostración construida con el skill `web-pages` de Claude Code.

**Stack**: HTML5 + CSS moderno (custom properties, grid) + JS vanilla. Sin build step.

**Publicada en**: https://eljack69.github.io/demo-landing/

## Estructura
```
.
├── index.html
├── assets/
│   ├── css/styles.css
│   ├── js/main.js
│   └── img/
├── favicon.svg
├── robots.txt
└── sitemap.xml
```

## Desarrollo local
Abrir `index.html` en el navegador. Sin servidor necesario.
